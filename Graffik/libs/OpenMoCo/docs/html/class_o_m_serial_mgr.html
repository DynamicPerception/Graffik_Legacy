Detailed Description

An OpenMoCo Serial Manager controls access to a particular QextSerialPort object, queueing commands when not connected and providing a mechanism for complete asynchronous communication with a MoCoBus device using QThread.

This class should not be used directly, instead the OMNetwork class should be used to manage busses and devices easily.
